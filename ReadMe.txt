In order to run the poject you have to follow these instructions:

1. If the "node_modules" folder is missing from your project
	you have open the terminal and run "npm install"
2. Open a terminal and write the following commands:
	"cd backend", "npm start"
3. Open a new terminal and write the following commands:
	"cd frontend", "npm start"
After running the "npm start"command in the frontend folder wait for
"Starting the develpoement server message".
It should take a couple of minutes for the message to show up.
